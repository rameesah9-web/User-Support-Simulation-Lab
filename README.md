# User Support Simulation Lab

A detailed Python program that simulates real first-line IT support work.

## What it does:
- Shows realistic business tickets for users **Adam**, **James**, and **Aria**
- Lets you view full ticket details (device, issue, symptoms)
- Step-by-step troubleshooting (restart, WiFi check, clear cache, password reset, etc.)
- Automatically logs every step and user feedback
- Generates a professional Support Log report (`My_Support_Log.txt`)

## Example Tickets:
- **Adam**: Internet keeps disconnecting on company Windows laptop
- **James**: Forgot Windows password on home PC
- **Aria**: Slow laptop after installing new software

## Why I built it:
To practise the exact workflow used in real IT Support roles: logging issues, troubleshooting logically, documenting steps, and resolving tickets.

## How to run:
1. Download the Python file
2. Run it with Python
3. Use menu option 2 to troubleshoot tickets and option 4 to generate the log

Built as part of my preparation for a Level 3 IT Apprenticeship.
